# U-IPTV 网络电视播放系统

U-IPTV 是一个现代化的网络电视播放平台，允许用户管理和播放各类IPTV频道。支持频道分类管理、在线状态检测、M3U文件导入导出等功能。

## 功能特性

- **频道管理** - 添加、编辑、删除和组织各类IPTV频道
- **在线状态检测** - 实时检测频道在线状态
- **M3U导入导出** - 支持从M3U文件导入频道列表，以及导出频道为M3U格式
- **频道分类** - 按照新闻、体育、电影、娱乐等分类管理频道
- **响应式播放器** - 自适应的视频播放器，支持播放控制
- **频道搜索** - 快速搜索和筛选频道
- **批量操作** - 支持批量选择和删除频道

## 技术栈

- **前端框架**: React 18
- **路由**: React Router
- **UI样式**: Tailwind CSS
- **状态管理**: React Hooks
- **视频播放**: HLS.js
- **构建工具**: Vite
- **图标库**: Lucide React
