# 键盘测试工具

键盘测试工具是一个现代化的键盘检测和测试平台，允许用户实时检测键盘按键状态、响应速度和连击性能。支持多种键盘布局，提供详细的按键分析功能。

## 功能特性

- **实时按键检测** - 实时显示按键状态，绿色表示正常按下，红色表示可能存在问题
- **响应速度测试** - 测量按键响应时间，帮助识别延迟问题
- **连击性能测试** - 检测按键连击性能，识别卡键问题
- **多键盘布局支持** - 支持标准104键、Mac键盘、游戏键盘、笔记本键盘等多种布局
- **故障按键识别** - 自动标记长时间未响应的按键
- **响应式设计** - 完美适配桌面、平板和手机设备

## 技术栈

- **前端框架**: React 18
- **UI样式**: Tailwind CSS
- **状态管理**: React Hooks
- **构建工具**: Vite
- **图标库**: Lucide React
