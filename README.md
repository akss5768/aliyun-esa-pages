# 云原生应用集合

欢迎来到我们的云原生应用集合项目！这里汇集了多个使用现代Web技术构建的单页应用程序，每个应用都专注于解决特定领域的问题。

## 项目概览

### U-Card - 卡牌收藏管理系统
一个现代化的卡牌收藏管理平台，允许用户创建、组织和管理各种类型的卡牌收藏。支持多种卡牌类型，提供AI辅助生成功能，以及本地数据存储能力。

### U-GameBBS - 游戏论坛系统
一个现代化的游戏论坛平台，允许玩家交流游戏心得、分享攻略、讨论游戏话题。支持帖子分类、评论互动等功能，以及本地数据存储能力。

### U-IPTV - 智能电视网络播放器
一个现代化的网络电视播放平台，允许用户在线观看IPTV直播流。支持频道管理、分类浏览、播放控制等功能，使用浏览器本地存储管理频道数据。

### U-Jobs - 招聘求职管理系统
一个现代化的招聘求职管理平台，允许企业发布招聘信息、求职者投递简历，以及职位管理功能。支持职位分类、在线申请等功能，以及本地数据存储能力。

### U-Keyboard - 键盘布局测试工具
一个实用的键盘布局测试工具，帮助用户测试和比较不同的键盘布局，支持多种布局选择和打字测试功能。

### U-Kol - 主播管理系统
一个现代化的主播管理平台，允许用户创建、组织和管理主播信息。支持主播信息展示、分类管理等功能，以及本地数据存储能力。

### U-Sample - 示例项目
一个示例项目，展示了基本的项目结构和开发模式，供开发者参考学习。

## 技术栈

所有项目均采用现代Web技术栈构建：

- **前端框架**: React 18
- **样式**: Tailwind CSS 或 Less
- **路由**: React Router
- **状态管理**: React Hooks
- **构建工具**: Vite 或 Webpack
- **数据存储**: 浏览器 localStorage（无需后端）

## 部署

所有应用都是单页应用(SPA)，可以轻松部署到任何静态托管服务，如：
- 阿里云ESA
- Vercel
- Netlify
- GitHub Pages
- 或任何支持静态文件托管的服务

## 特点

- **离线可用**: 所有数据存储在浏览器本地，无需网络连接即可使用
- **轻量级**: 每个应用都经过优化，加载速度快
- **响应式设计**: 完美适配桌面、平板和移动设备
- **易于部署**: 作为静态文件部署，无需复杂的服务器配置
